# Agam Pathak Portfolio

Personal portfolio built with Next.js 16, React 19, Tailwind CSS 4, and Framer Motion.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Structure

- `src/app`: landing page, resume page, and project case studies
- `src/components`: shared UI, motion, theme, analytics, and contact form components
- `public/assets`: logos and screenshots used across the portfolio

## Notes

- Remote skill icons are loaded from `https://skillicons.dev`.
- Contact form submissions are forwarded through FormSubmit.
